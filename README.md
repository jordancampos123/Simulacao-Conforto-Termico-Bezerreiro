# Simulação do conforto térmico de um bezerreiro utilizando o EnergyPlus

Este repositório reúne os arquivos utilizados nas simulações computacionais apresentadas no artigo:

**"Análise do conforto térmico de um bezerreiro utilizando ventilação natural por meio do software EnergyPlus".**

## Conteúdo

### model_inputs
- `bezerreiro.idf` — Modelo computacional desenvolvido no EnergyPlus 8.7.
- `RS_Pelotas.epw` — Arquivo climático utilizado nas simulações.

### model_output
- `bezerreiro.csv` — Resultados exportados das simulações.
- Arquivos auxiliares gerados automaticamente pelo EnergyPlus.

## Software utilizado

- SketchUp 2017
- EnergyPlus 8.7

## Objetivo

Disponibilizar os arquivos utilizados na modelagem computacional, permitindo a reprodução das simulações apresentadas no artigo e contribuindo para a transparência e a reprodutibilidade da pesquisa.
